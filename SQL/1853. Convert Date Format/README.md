# [DAYNAME() Function](https://dev.mysql.com/doc/refman/8.0/en/date-and-time-functions.html#function_dayname)

**DAYNAME()** function return the name of the weekday

```
SELECT DAYNAME('2007-02-03');         -> 'Saturday'
```
